## Installation & Deployment üì¶
- Clone the repository and modify the contents of <b>index.html</b> according to your requirements.
- Add or remove images from the `nickcapobianco.github.io/assets/img/` directory per your requirements.
- I highly recommend using [Github Pages](https://create-react-app.dev/docs/deployment/#github-pages) to deploy the website the EASIEST WAY.
- To deploy your website, first you need to create a github repository with the name `<your-github-username>.github.io`. Don't use any other name as this is the website link generated by GitHub.
- <b>NOTE:</b> Make sure to set your `analyticsId` from your Google Analytics account inside the Google Analytics script tag if you want to use your own Google Analytics account.
- If you don't have a Google Analytics account, follow this link: [![Google Analytics]](https://support.google.com/analytics/answer/9304153?hl=en#zippy=%2Cadd-your-tag-using-google-tag-manager)
